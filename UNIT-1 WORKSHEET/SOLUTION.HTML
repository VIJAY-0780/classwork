Q1. Introduction to HTML & its features

Definition: HTML (HyperText Markup Language) is the standard language used to structure and present content on the web.

  • Tags: Uses elements like headings, paragraphs, and images to define content.
  • Structure: Provides semantic layout for organized, meaningful pages.
  • Media & links: Supports hyperlinks, images, audio, and video.

Q3: Semantic vs Non-Semantic Elements

Difference: Semantic elements clearly describe their meaning and role in the document (e.g., <header>, <article>), while non-semantic elements (e.g., <div>, <span>) do not convey specific meaning and are used mainly for layout or grouping.

  • Semantic examples: <header>, <footer>
  • Non-semantic examples: <div>, <span>
